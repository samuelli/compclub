Arduino workshops

Computing Club is running free Arduino-based workshops at UNSW with the aim to teach basic programming techniques. An Arduino is essentially a small, portable computer the size of a credit card. It can take in inputs, using things like a light sensor, and produce outputs, such as blinking an LED light.

This is a hands-on workshop which allows students to learn the basics of computing in a fun, practical way. Students will learn programming basics by programming an Arduino, including:

  • How to control LEDs
  • How to use the serial communication bus
  • How to use sensors such as photocells
  • How to play music using a buzzer

No assumed knowledge is required for this workshop and equipment is provided by Computing Club. However, the workshop is targeted at students in years 9-12.
